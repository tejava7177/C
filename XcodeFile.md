Xcode 파일을 입출력 하는 방법


1. Xcode 파일을 정상적을 생성한다.
2. 읽어올 텍스트 파일을 생성한다.
3. newfile - empty 파일을 .txt로 확장자를 변경한다.
************************
4. 이후 Xcode 설정을 변경한다.
  ->메뉴바에서 설정하느 과정
  : product --> scheme --> Edit scheme에 Working Directory를 체크하고
    해당 xcode 파일 경로르 설정한다.
************************

이후 정상적을 파일이 생성되었는지 확인한다.
